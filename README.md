# A Robust Unsupervised Ensemble Pipeline with explainable AI for Stellar Anomaly Detection in GAIA DR3

## Project Overview
This project performs exploratory data analysis and preprocessing on a stellar dataset using Python. The notebook cleans the data, engineers useful features, and prepares it for further analysis and machine learning.

## Features
- Load and inspect the dataset
- Handle missing values
- Remove invalid records
- Feature engineering
  - Color Index
  - Total Proper Motion
  - Distance (Parsecs)
- Exploratory Data Analysis (EDA)
- Data visualization

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## How to Run

1. Clone the repository

```bash
git clone https://github.com/anikals375/Stellar-Analysis.git
```

2. Install the required libraries

```bash
pip install -r requirements.txt
```

3. Open the notebook

```bash
jupyter notebook
```

4. Run `FINAL_Stellar_Analysis.ipynb`
